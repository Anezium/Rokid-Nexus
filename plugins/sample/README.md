# Sample plugin

The minimal, copyable reference for building a Nexus plugin: one headless
manifest, one `NexusPluginService` showing a card and reacting to input, one
kit-based settings screen. Start here, then read
[docs/PLUGINS.md](../../docs/PLUGINS.md) for the structure and design-kit
rules and [docs/PLUGIN_SDK.md](../../docs/PLUGIN_SDK.md) for the SDK
reference.
